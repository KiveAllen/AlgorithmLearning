# 冒泡排序

### bubbleSort.java

```java
package com.xiaoyou.算法;

public class bubbleSort {
    //冒泡排序
    public static void doBubbleSort(int[] a) {
        //一共需要length-1趟
        for (int i = 0; i < a.length - 1; i++) {
            for (int j = 0; j < a.length - i - 1; j++) {
                if (a[j] > a[j + 1]) {
                    int t = a[j + 1];
                    a[j + 1] = a[j];
                    a[j] = t;
                }
            }
        }
    }
}
```

### 优化冒泡 optimizeBubble.java

```java
public class optimizeBubble {
    public static void optimizeBubble(int[] a) {
        int length = a.length;
        boolean f = true;//做标记
        while (f)//当没发生数字调换则直接退出程序
        {
            f = false;//重置f
            for (int i = 0; i < length-1; i++) {
                if (a[i] > a[i + 1]) {
                    int t = a[i];
                    a[i] = a[i + 1];
                    a[i + 1] = t;
                    f = true;若发生改变，f赋值为true
                }
            }
            length--;
        }
    }
}
```
